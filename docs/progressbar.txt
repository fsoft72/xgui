Progressbar
	derives from Widget

METHODS

*
int getPos ()
	- returns the current position of the progress counter

*
void setPos ( int pos )
	- sets the current postition of the progress counter



PROPERTIES

+
pos (RW)
	- sets or retrieves the current position of the progress counter

+
min (RW)
	- sets or retrieves the starting position of the progress counter
	  default: "0"

+
max (RW)
	- sets or retrieves the maximum position of the progress counter
	  default: "100"


EVENTS

@
onchange (linked to xgui::TextCallback)
	- fired when the position of the progress counter changes.
	  the string parameter passed to the callback function is the current progress counter position.
