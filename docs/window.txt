VBox
	derives from Container

METHODS

*
void show()
	- shows the window and all of its children

*
int modalRun()
	- shows the window as a modal window (i.e:. this method doesn't return until the window is destroyed).
	  The return value is the value passed to the modalReturn() method.
	
*
bool isModal()
	- returns true if the window is a modal; false otherwise.

*
void modalReturn(int response)
	- exits from a modal loop, closes the related window and returns the <response> value as the return value of the modalRun() method.
	

*
xgui::Widget * getChild()
	- returns the window child. Same as getChild(0).
	

*
xgui::Menu * getMenu()
	- returns the menu assigned to the window, if one exists.


*
void maximize()
	- maximizes the window.


*
void minimize()
	- minimizes the specified window and activates the next top-level window in the z-order


*
void restore()
	- activates and displays the window.
	  If the window is minimized or maximized, it will be restored it to its original size and position.

*
void setIcon ( Image * img )
	- sets the image displayed in the window title bar.

*
xgui::Image * getIcon()
	- returns the image displayed in the window title bar, or NULL if no image is displayed.


PROPERTIES

+
text (RW)
	- sets or retrieves the text displayed on the window title bar.

+
size (RW)
	- sets or retrieves the window dimensions in the format "width height".

+
position (RW)
	- sets or retrieves the window position in the screen in the format "x y".
	  If set to "center" then the window is positioned at the center of the screen.

+
resizable (RW)
	- if set to "1" the window has a sizing border; if set to "0" the window can't be resized.

+
border (RW)
	- if set to "1" the window has a border and a title bar; if set to "0" the window doesn't have
	  any border or title bar.

+
status-text (RW)
	- if set to a non-empty string then a status bar with the specified text is displayed at the bottom of the window.

+
icon (RW)
	- sets or retrieves the path of the file of the image displayed on the window title bar. See also the setIcon() method.


EVENTS

@
onclose (linked to xgui::Callback)
	- fired when the window is about to be closed.
	  Returning xgui::EVT_BLOCK from the callback function will prevent from closing the window.


PUSHED PROPERTIES (properties registered on all children)
	none.
