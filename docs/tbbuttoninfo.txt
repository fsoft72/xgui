TbButtonInfo

DESCR
	Information class used to describe a button of the Toolbar widget.


CONSTRUCTORS

*
TbButtonInfo()
	- initializes a new empty button

*
TbButtonInfo ( const std::string & cname, const std::string & ctext, xgui::Image * cimage, const std::string & ctooltip )
	- initializes a new tollbar button with the specified information. See the member variables description.


MEMBER VERIABLES

%
std::string name
	- the name of the button (used to refer to the button in Toolbar methods)


%
std::string text
	- the text displayed over the button


%
xgui::Image * image
	- the image displayed over the button


%
std::string tooltip
	- the tooltip text displayed for the button
