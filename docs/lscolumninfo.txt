LsColumnInfo

DESCR
	Information class used to describe a column for the List widget.


CONSTRUCTORS

*
LsColumnInfo()
	- initializes a column with a width of 100 pixels.

*
LsColumnInfo ( int cpos, const std::string & cid, const std::string & cname, int cwidth = 100, const std::string & calign = "left" )
	- initializes a column with the specified information. See the member variables description.


MEMBER VERIABLES

%
int pos
	- the order position of the column in the List widget

%
std::string name
	- the column name displayed when column headers are visible

%
std::string id
	- the column id


%
int width
	- the column width in pixels

%
std::string align
	- the alignment of the text in the column; it can be one of the following values:
		"left", "right", "center"
