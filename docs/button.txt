Button
	derives from Widget

DESCR
	The Button is a widget that the user can click to provide input to the application.
	This class cannot be directly instantiated, but must be created using the
	Master::CreateButton() function.

METHODS


*
int setIcon ( Image * img )
	- sets the icon to be displayed on the button

*
xgui::Image * getIcon()	
	- returns the image displayed on the button


PROPERTIES

+
image-path (RW)
	- sets or retrieves the path of the file containing the image displayed on the button;
	  if set to empty string no image is displayed


+
text (RW)
	- sets or retrieves the text displayed on the button

+
pressed (RW)
	- if set to "1" the button is in a pressed state (for toggle or radio buttons), if set to "0" the button is released

+
alignment (RW)
	- sets or retrieves the alignment of the button text; it can be one of the following values:
		"top-left", "top", "top-right", "left", "center", "right", "bottom-left", "bottom", "bottom-right"
	  default: "center"


EVENTS

@
onclick (linked to xgui::Callback)
	- fired when the button is clicked
