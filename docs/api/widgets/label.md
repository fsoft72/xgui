# Label

The `Label` widget displays static text. It derives from `Widget` and provides a simple way to display non-editable text in the user interface.

## Class Hierarchy

```
Object
└── Widget
    └── Label
```

## Methods

This widget has no additional methods beyond those inherited from `Widget`.

## Properties

### `text` (Read/Write)

Sets or retrieves the text displayed in the label.

**Type:** String

**Default:** `""`

---

### `alignment` (Read/Write)

Sets or retrieves the text alignment of the label.

**Values:**
- `"left"` - Left-aligned text
- `"center"` - Center-aligned text
- `"right"` - Right-aligned text

**Default:** `"left"`

---

## Events

This widget has no events beyond those inherited from `Widget`.
