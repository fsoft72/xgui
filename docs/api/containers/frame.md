# Frame

The `Frame` widget is a container that displays a decorative border with an optional text label. It derives from `Container` and is commonly used to visually group related widgets.

## Class Hierarchy

```
Object
└── Widget
    └── Container
        └── Frame
```

## Methods

This widget has no additional methods beyond those inherited from `Container`.

## Properties

### `text` (Read/Write)

Sets or retrieves the text displayed on the frame border.

**Type:** String

**Default:** `""`

---

### `alignment` (Read/Write)

Sets or retrieves the alignment of the text displayed on the frame border.

**Values:**
- `"left"` - Left-aligned text
- `"center"` - Center-aligned text
- `"right"` - Right-aligned text

**Default:** `"left"`

---

## Events

This widget has no events beyond those inherited from `Widget`.

## Pushed Properties

This widget does not register any properties on its children.
