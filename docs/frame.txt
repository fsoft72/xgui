Frame
	derives from Container

METHODS
	none.


PROPERTIES

+
text (RW)
	- sets or retrieves the text displayed on the frame border.

+
alignment (RW)
	- sets or retrieves the alignment of the text displayed on the frame border.
	  It can be one of the following strings: "left, "center", "right".
	  default: "left"


EVENTS
	none.


PUSHED PROPERTIES (properties registered on all children)
	none.
