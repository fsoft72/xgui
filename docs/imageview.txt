ImageView
	derives from Widget

METHODS
			xgui::Image * getImage();
			void setImage ( xgui::Image * img );
			std::pair<int, int> getViewSize();
			bool isScrolled();


*
xgui::Image * getImage()
	- returns the image that the ImageView widget is currently displaying

*
void setImage ( xgui::Image * img )
	- sets a new image to display in the ImageView widget


*
std::pair<int, int> getViewSize()
	- returns a pair of integers representing the view size of the image
	  (the view size represents the visible portion of the image)

*
bool isScrolled()
	- returns true if the image is scrolled, false if the image is resized to fit the view size.
	  see also the "scrolled" property.


PROPERTIES

+ scrolled (RW)
	- if set to "1" the image is scrolled: only the portion of the image specified the "view-size" property
	  is displayed and scrollbars have to used to scroll the image.
	  if set to "0" the image is resized to fit the dimensions specified with the "view-size" property.

+ view-size (RW)
	- sets or retireves the view size. The view size represents the visible portion of the image.


+ image-path (RW)
	- sets or retrieves the path of the file containing the currently displayed image.


+ alignment (RW)
	- sets or retrieves the image alignment in the imageview widget.
	  It can be one of the follwing strings: "top-left", "top", "top-right", "left", "center", "right",
	  "bottom-left", "bottom", "bottom-right".
	  default: "center"


EVENTS
	none.
