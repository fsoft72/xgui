Menu
	derives from Widget

METHODS

*
void addItem(std::string const &parent_id, std::string const &id, std::string const &text, xgui::Image * icon = 0, std::string const & type = "" )
	- adds a new menu item. <parent_id> specifies the id of the parent menu item, or empty string if the item is to be added at to root level.
	  <id> is an identification string assigned to the new menu item. <text> is the displayed menu item text. <icon> is an optional
	  pointer to an image to be displayed near the mneu item text. <type> can be an empty string for a standard menu item, or one of the following
	  values:
	  	"separator"	-> creates a line separator
		"check"		-> creates a menu item which can be checked/unchecked (the item will have a small icon that marks the check state)
		"radio_begin"	-> pseudo-item (not visible) that marks the beginning of a radio item group
		"radio_end"	-> pseudo-item (not visible) that marks the end of a radio item group

*
void delItem(std::string const &id)
	- removes the specified menu item


*
void popupMenu(std::string const &id)
	- displayes the popup-menu identified by <id> at the current mouse position.


*
void enableItem(std::string const &id)
	- enables the specified menu item (the item can be clicked or selected)


*
void disableItem(std::string const &id)
	- disables the specified menu item (the item can't be clicked or selected and is displayed in gray)


*
bool isItemEnabled(std::string const &id)
	- returns true if the specified item is enabled, false otherwise.


*
void checkItem(std::string const &id)
	- marks the specified checkable menu item as checked


*
void uncheckItem(std::string const &id)
	- removes a check from the checkable checkable menu item

*
bool isItemChecked(std::string const &id)
	- returns true if the specified menu item is checked, false otherwise


*
void setItemText(std::string const &id, std::string const &text)
	- changes the text of the specified menu item


*
std::string getItemText(std::string const &id)
	- returns the text of the specified menu item


PROPERTIES
	none.


EVENTS

@
onselect (linked to xgui::TextStatusCallback)
	- fired when the user clicks on a menu item
	  the string parameter of the callback function represents the id of the clicked menu item.
	  The status parameter is set to 1 if the clicked menu item has been checked, otherwise is set to 0.
