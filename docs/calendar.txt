Calendar
	derives from Widget


DESCR
	Calendar widget is used to display a calendar-like interface to input dates.
	This class cannot be directly instantiated, but must be created using the
	Master::CreateCalendar() function.


METHODS
	(none)


PROPERTIES

+
day (RW)
	- sets or retrieves the day of the month currently selected in the calendar

+
month (RW)
	- sets or retrieves the month currently selected in the calendar

+
year (RW)
	- sets or retrieves the year currently selected in the calendar

+
date (RW)
	- sets or retrieves the full date currently selected in the calendar
	  the format is "year-month-day"


EVENTS

@
onselect (linked to xgui::TextCallback)
	- fired when the user selects a date
	  the string parameter specifies the selected date in the format "year-month-day"
